Objective:
-three view modes:
 1)overhead orthogonal
 2)overhead perspective
 3)first person
-3D objects

Description: 
3D Scene of cylinders with roofs, square houses, and more cylinders.
Added different view modes.

Key Bindings:
 a		toggle axes
 0		reset view angle
 m		toggle orthogonal/perspective
 arrows		change view angle
 [		decrease field of view
 ]		increase field of view
 n		toggle first person
 arrows		navigate first person
 ESC		exit
